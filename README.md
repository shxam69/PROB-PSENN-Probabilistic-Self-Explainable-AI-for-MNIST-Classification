# Prob-PSENN

Prob-PSENN is a probabilistic self-explainable AI project for MNIST digit classification using a prototype-based neural network, uncertainty analysis, and a Gradio-based interactive demo.

## Features
- Prototype-based digit classification
- Uncertainty estimation
- Explainability visualizations
- Calibration analysis
- Gradio web interface
- MNIST training and evaluation

## Project Files
- `PROBALISTIC_perfect_setup-1.ipynb` — main notebook
- `requirements.txt` — dependencies
- `.gitignore` — ignored files
- `model.pth` — saved model weights, if included
- `app.py` — standalone app, if separated from the notebook

## How to Run
1. Install the dependencies from `requirements.txt`.
2. Open the notebook or run the app.
3. Train or load the model.
4. Draw a digit and view the prediction with explanations.

## Goal
To explore interpretable deep learning with prototype reasoning, uncertainty analysis, and explainable AI visualizations.
